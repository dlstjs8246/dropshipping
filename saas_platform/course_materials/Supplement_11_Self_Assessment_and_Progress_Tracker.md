# 📋 Supplement 11: 자가 진단 & 학습 진척 추적 시스템

> **목적**: 14주 + AI Mastery Track L1/L2/L3 + 운영 부록 전체를 가로지르는 **단일 학습 진척 추적 시스템**. "지금 내가 어디 있나?"를 객관 진단하는 메타 자료.
>
> **사용법**: 매주 일요일 22:00 본 문서를 펼쳐서 (1) 이번 주 체크포인트 답변, (2) 다음 주 준비 점검. 강의 시작 전 1번 + 매주 1번 + 졸업 시 1번 = 총 16번 사용.
>
> **소요 시간**: 첫 사용 30분 (Pre-flight) + 매주 10분 + 졸업 60분.
>
> **연계**: 본 자료는 **타 자료를 통합하는 인덱스** — 진단 결과별로 어느 자료를 봐야 할지 안내합니다.

---

## 0. 본 시스템의 구조

```
[Week 0] Pre-flight 체크리스트 (20문항)
         "강의 시작 자격이 되는가?"
              ↓
[Week 1~14] Weekly Checkpoint × 14
         매주 5문항 = 총 70문항
         각 주차 종료 후 5분 작성
              ↓
[L1·L2·L3 졸업 자가 진단] 통합
         Supplement 08·09·10의 자가 진단을
         하나의 트래커로 집계
              ↓
[14주 졸업 루브릭] 3축
         매출 / 시스템 / AI 활용
         3축 모두 PASS = 진정한 졸업
              ↓
[30일 / 90일 / 1년 후 재진단]
         시스템이 잘 도는가?
```

## 4. 14주 졸업 루브릭 — 3축

3축 모두 **PASS** = 진정한 졸업. 1축이라도 미달 시 다음 90일 ([Appendix D](./Appendix_D_First_100_Customers.md)) 더 운영.

### 🔴 1축: 매출 (Revenue Track)

| 등급 | 기준 | 의미 |
|---|---|---|
| **A** | 누적 50명+ / 매출 $2,000+ | 검증 완료, Stage C 진입 |
| **B** | 누적 10~49명 / 매출 $400~$2K | 검증 중, 90일 추가 운영 |
| **C** | 누적 1~9명 | Seed 단계, [Appendix D Stage A](./Appendix_D_First_100_Customers.md) 재실행 |
| **F** | 매출 0건 | 14주 학습 OK, 90일 더 필요 |

### 🟢 2축: 시스템 (System Track)

| 등급 | 기준 |
|---|---|
| **A** | Make 시나리오 3개+ / Klaviyo 시퀀스 / Track123 / RAG CS 봇 모두 가동 |
| **B** | Make 1~2개 + Klaviyo 1개 |
| **C** | Make 1개만 |
| **F** | 자동화 0 |

### 🟣 3축: AI 활용 (AI Mastery Track)

| 등급 | 기준 |
|---|---|
| **A** | L1+L2 졸업 + L3 Build 1개+ 가동 |
| **B** | L1+L2 졸업 |
| **C** | L1만 졸업 |
| **F** | L1 미통과 |

### 종합 판정

```
3축 모두 A → 🏆 졸업 + 1기 Lifetime Free 베타 + 추천서
3축 모두 B+ → ✅ 졸업 + 90일 멘토링
1축이라도 C+ → 🟡 부분 졸업 + 90일 추가
1축이라도 F → ❌ 재이수 권장 (해당 축 보강)
```

### 사용법 (매주 일요일 22시)

1. 본 §1~§5를 작성 (10분)
2. 자유 글쓰기 200자 — "지금 가장 답답한 것"
3. 위 200자 + 본 시스템 데이터를 [Master Prompts §38 (멘탈 코치)](./02_Master_Prompts.md)에 입력
4. AI가 **빠진 함정 + 객관 진단 + 다음 7일 5개 행동 + 위로** 자동 출력
5. 출력의 행동 5개 중 1개라도 **24h 이내** 실행

> ⚠️ **AI는 코치 보조**: 진짜 위기 (자살 충동·우울증) = 즉시 전문가. 본 §38은 사업적 함정 진단 한정.

## 7. 본 시스템 자가 점검 — "이걸 쓰고 있는가?"

본 자가 진단 시스템을 **매주 일요일 22시 정확히** 작성하지 않으면 의미 없음. 본인 사용도 점검:

- [ ] 매주 일요일 22시 알람 설정 (휴대폰)
- [ ] 본 문서를 노션 사이드바 또는 즐겨찾기에 등록
- [ ] 매주 작성 시간 평균 10분 이내
- [ ] 미달 항목 발견 시 다음 주 보강 액션 메모
- [ ] 4주 연속 작성 = 습관 형성

4주 연속 작성 → 학습 습관 정착. 1년이면 52번 자기 진단 = 어떤 강의보다 강력한 자기주도 학습.
