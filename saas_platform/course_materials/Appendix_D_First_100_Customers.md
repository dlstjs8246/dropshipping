## 📉 0. "무매출의 골짜기" — 왜 첫 100명이 가장 힘든가

신규 셀러의 95%가 **첫 매출 30일 안에 그만둡니다**. 이유는 단순합니다.

| 단계 | 심리 상태 | 흔한 오류 |
|---|---|---|
| Day 0~7 | "오픈했으니 누군가 오겠지" | 트래픽이 0인 걸 모름 |
| Day 8~14 | "광고를 더 태워야 하나?" | 위닝 검증도 안 됐는데 광고비 소진 |
| Day 15~30 | "상품이 별로인 것 같다" | 상품 7번 갈아치움 (실패 누적) |
| Day 31~60 | "내가 안 맞나봐" | 계정 정리하고 떠남 |

> **핵심 진실**: 첫 100명은 **광고로 사오는 게 아니라**, **"이미 검증된 상품 + 신뢰 시그널 3개" 를 바닥에 깐 뒤** 오가닉 + 소액 광고로 흘려보내는 겁니다.

## 🌱 2. Stage A: Seed — Day 0~14 (목표: 첫 매출 1명)

광고 없이 **첫 1명**을 만드는 단계입니다. 이 1명이 모든 후속 단계의 신뢰 시그널을 만듭니다.

### A-1. Day 0 체크리스트 (오픈 직전 — 절대 생략 금지)

- [ ] **상품 페이지 4개 이상** (위닝 후보 1~2개 + 보조 상품 2~3개)
- [ ] **상세페이지 PAS 카피** ([Master Prompts §2](./02_Master_Prompts.md))
- [ ] **상품 스코어카드 23점 이상** ([Week 3](./Week03_Landed_Cost_Scorecard.md))
- [ ] **Bogus Gateway 결제 테스트 완료** ([Week 6](./Week06_Cursor_Coding_Optimization.md))
- [ ] **Track123 배송 추적 페이지 메뉴 등록** ([Week 6](./Week06_Cursor_Coding_Optimization.md))
- [ ] **About Us 페이지** (얼굴 사진 1장 + 창업 이유 200자) ← 신뢰 시그널 1
- [ ] **환불 정책 페이지** ([Appendix A](./Appendix_A_Refund_Legal_Checklist.md)) ← 신뢰 시그널 2
- [ ] **연락처 (이메일 + 폼) 명시** ← 신뢰 시그널 3

> **ChatGPT 잘못된 조언 주의**: "트래픽 1만 명 모은 뒤 오픈하라"는 SNS 흔한 조언은 무시하세요. 빈 스토어를 보여주지 않는 것보다 **빈 스토어를 빠르게 보여주고 피드백을 받는 게 1.5배 빠릅니다**.

### A-2. Day 1~3: 지인 시드 7명 (Friends & Family Seed)

> **목표**: 진짜 결제 1건 + 진짜 후기 3건 = 신뢰 시그널 4

신규 스토어의 **0% 후기 페이지**가 가장 큰 이탈 원인입니다. 지인 7명에게 직접 물어보세요. 단 **무료 증정 금지**(이건 후기 가치를 0으로 만듭니다). 대신:

```
[지인용 메시지 템플릿 — 카톡으로 그대로 복사]

안녕 [이름]아, 나 OO 비즈니스 시작했어. 첫 매출 도와줄 수 있어?
- 상품: [URL]
- 가격: $XX (정가 그대로, 할인 X)
- 부탁: 정상 결제 후 솔직한 후기 1줄 (좋아도 나빠도 진짜 의견)

내 사업 시작 도와주는 거니까 정가에 사주는 게 진짜 도와주는 거야.
구매 후 영수증 사진 한 장만 보내줘. 내가 첫 5명한테 [수공예 감사 카드 / 다음 신상 30% 코드 / 직접 손글씨 편지] 줄게.
```

**필수 규칙**:
- 결제는 **반드시 본인 카드로 정상 결제** (가족 카드 X — Stripe Dispute 위험)
- 후기는 **단점도 포함된 솔직한 1~2줄** (5점 ★★★★★ 만점만 있으면 가짜로 보임)
- 영상 후기는 **1명만** (나머지는 텍스트만)

### A-3. Day 1~14: 무료 오가닉 콘텐츠 5개

광고비 0원으로 **틱톡/릴스 5개**를 발행합니다. 모두 **본인 휴대폰으로 15초**.

| # | 콘텐츠 타입 | 핵심 메시지 | Hook (0~3초) |
|:--:|---|---|---|
| 1 | Unboxing | 박스에서 처음 꺼낼 때 | "I just got the [상품]…" |
| 2 | Problem reveal | 평소 겪던 문제 시연 | "POV: 매일 [고통] 겪는 너" |
| 3 | Before/After | 사용 전후 시각 차이 | "Watch what 30 seconds does" |
| 4 | Behind the scenes | 창업 동기 | "Why I started [브랜드]" |
| 5 | Q&A | 자주 묻는 질문 | "Top question I get…" |

> **핵심**: [Week 7](./Week07_AI_Creative_Factory.md) 의 3-요소 (Hook-Body-CTA) 구조를 그대로 따릅니다. **CTA 없이는 발행 금지** — "프로필 링크 → 50% 첫 구매 코드" 한 줄이라도 박아넣으세요.

### A-4. Day 14 게이트 — 다음 단계 진입 조건

- [ ] **결제 1건 이상** (지인이어도 OK — 신뢰 시그널 누적)
- [ ] **후기 3건 이상** (★★★★★ 5점 만점만이면 1건은 4점으로 수정)
- [ ] **틱톡/릴스 5개 발행 완료**
- [ ] **노출 합계 1,000회 이상** (각 영상 200+)

> 실패 시: [Recovery Tree #1](#case-1-day-14에-매출-0건) 으로 이동.

## 🧪 3. Stage B: Validate — Day 15~30 (목표: 누적 10명)

지인을 넘어 **모르는 사람 1명**을 만드는 단계. **"이게 진짜 비즈니스인가?"** 의 분기점.

### B-1. UGC 확보 — 진짜 후기 영상 1개

지인 시드 5명 중 **가장 좋아한 1명**에게 영상 후기를 부탁합니다.

```
[UGC 부탁 템플릿]

[이름]아, 상품 잘 쓰고 있다고 했지? 영상 후기 30초만 찍어줄 수 있어?
- 형식: 휴대폰 세로 영상, 너 얼굴 + 상품 같이 등장
- 내용: "이거 써보니까 [구체적 변화] 됐어"
- 보상: 다음 신상 무료 1개 또는 [감사 표시]
```

받은 영상은 **CapCut으로 자막 박고 틱톡 6번째 영상으로 발행**. 이게 **첫 진짜 사회적 증거**입니다.

### B-2. 추천 코드 시스템 — Tapcart / Goaffpro

지인 5명에게 **개인 추천 코드** 를 발급합니다 (Goaffpro 무료 플랜).

| 코드 | 친구 가격 | 지인 보상 |
|---|---|---|
| `MIN10` (지인 이름) | 10% 할인 | 친구 결제 시 10% 적립 |

> **효과**: 지인이 자기 친구한테 자발적으로 코드 뿌립니다. **2차 확산 시작점**.

### B-3. 첫 모르는 사람 매출 — Day 18~25

오가닉 영상 5~10개가 누적되면 **노출 5,000~10,000회 구간**에서 모르는 사람 1명이 결제합니다. 이때:

- [ ] **즉시 감사 이메일** ([Master Prompts §11](./02_Master_Prompts.md) 환영 시퀀스 §1)
- [ ] **결제 후 24시간 내 후기 요청 메일** (자동화 — [Supplement 01](./Supplement_01_Email_Automation_Playbook.md))
- [ ] **배송 직후 (Track123) 영상 후기 인센티브** ($5 코드 또는 5% 적립)

### B-4. Day 30 게이트

- [ ] **누적 10명 매출** (지인 7 + 모르는 사람 3 이상)
- [ ] **★★★★★ 후기 5건 + ★★★★ 1건** (믹스가 진짜로 보임)
- [ ] **UGC 영상 1개** (틱톡 발행)
- [ ] **CVR 1.5% 이상** (방문자 100명 중 1.5명 결제)
- [ ] **노출 합계 20,000회 이상**

> 실패 시: [Recovery Tree #2](#case-2-day-30에-누적-매출-3건-이하) 로 이동.

## 🚀 4. Stage C: Accelerate — Day 31~60 (목표: 누적 50명)

오가닉으로 검증된 영상 **1~2개를 광고로 부스팅**하는 단계.

### C-1. 오가닉 폭발 영상 식별 — 광고 진입 조건

다음 **3가지 모두 만족**하는 영상에만 광고비를 태웁니다.

| 조건 | 임계치 | 의미 |
|---|---|---|
| 노출 | 5,000+ | 알고리즘이 좋아함 |
| 완주율 | 30%+ | Hook이 작동함 |
| 댓글 + 저장 | 합계 50+ | 의도된 관심 |

> 셋 중 하나라도 안 되면 **광고 금지**. [Week 9](./Week09_SparkAds_and_Organic.md) 의 BEP CPA 매트릭스 적용.

### C-2. Spark Ads 진입 — Day 35~40

선정된 영상에 **하루 $5**를 7일간 태웁니다 (총 $35).

```
[광고 세팅 핵심]
- 캠페인 목표: Conversion (Complete Payment)
- 타겟: 미국 18~44 + 관심사 [내 니치]
- 일예산: $5
- 영상: 오가닉 폭발 영상의 Spark Code
- KPI: ROAS 1.5+ (1주차) → 2.0+ (2주차)
```

> **Kill Rule**: 3일 연속 ROAS < 1.0 이면 즉시 광고 중지. 다른 영상으로 갈아치우기. 절대 "조금만 더 가보자" 금지.

### C-3. 재구매 시퀀스 가동 — Klaviyo 마이그레이션 트리거

월 매출 $3K 돌파하거나 누적 30명 넘으면 [Supplement 01](./Supplement_01_Email_Automation_Playbook.md) 의 Klaviyo 마이그레이션 진행. 이 시점부터 **5통 환영 + 3단 장바구니 이탈** 자동 발송.

### C-4. Day 60 게이트

- [ ] **누적 50명 매출**
- [ ] **CVR 2%+**
- [ ] **광고 ROAS 1.5+** (3일 평균)
- [ ] **이메일 구독자 200+** (즉시 자산이 됩니다)
- [ ] **재구매 1명 이상** (LTV 입증)

> 실패 시: [Recovery Tree #3](#case-3-day-60에-roas-10-아래) 로 이동.

## 📈 5. Stage D: Scale — Day 61~90 (목표: 누적 100명)

광고 + 제휴 동시 가동 단계. 첫 100명을 채우면 **다음 100명은 4배 빠릅니다**.

### D-1. 광고 스케일업 — $5 → $20/일

[Week 9](./Week09_SparkAds_and_Organic.md) 의 스케일업 규칙을 따릅니다. 핵심:

| 기존 | 신규 | 조건 |
|---|---|---|
| $5/일 | $10/일 | 3일 평균 ROAS 2.0+ |
| $10/일 | $20/일 | 7일 평균 ROAS 2.0+ |
| 더 이상 X | $20에서 멈춤 | 5일 ROAS 1.5~2.0 횡보 |

## 🎓 9. 졸업 후 — 100명 다음 100명

첫 100명을 채우면 다음 단계는 **3가지 동시 가동**입니다.

1. **상품 다각화** — 위닝 1개 검증됨 → 보조 상품 3~5개 라인업 ([Week 12](./Week12_Data_Driven_ScaleUp.md) 매트릭스로 결정)
2. **광고 채널 다각화** — TikTok 단일 의존 X → Meta Ads 추가 (틱톡 알고리즘이 본인 니치를 안 띄울 리스크)

## ⚠️ 10. 5가지 절대 금지 사항

1. ❌ **자기 카드로 자기 스토어 결제 (Wash trade)** — Stripe·Shopify 즉시 정지. 가족 카드도 X
2. ❌ **★★★★★ 5점 만점 후기만** — 가짜로 보여 CVR 떨어짐. ★4 1개는 필수
3. ❌ **검증 없이 광고비 $50/일** — 첫 검증 없이 광고는 거의 항상 마이너스. **Spark Ads = 검증 후 부스팅 도구**
4. ❌ **상품 7번 갈아치우기** — 한 상품 30일 풀 데이터 본 뒤 KILL. "어쩌다 잘 될 것 같아서" 갈아치우기는 학습 0
5. ❌ **"광고비 좀만 더 태우면 될 것 같아"** — ROAS 1.0 미만에서 이건 항상 거짓말. 즉시 중지
