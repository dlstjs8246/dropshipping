## 워크플로우 사고법

> **단일 AI 한계**: Claude는 검색이 약하고, GPT는 긴 문맥이 약하고, Perplexity는 창작이 약합니다.
> **다중 AI 파이프라인**: 각 AI의 강점만 골라 출력을 다음 AI의 입력으로 던지면 → **품질 + 비용 + 속도 모두 최적화**.

## 🥬 레시피 1: 니치 발굴 4단 파이프라인 (W2)

```
[Perplexity]    → [Claude]      → [Claude]       → [Gemini]
실시간 트렌드     7대 필터 평가     스코어카드        시각화 차트
   │                  │                  │                  │
   ▼                  ▼                  ▼                  ▼
"2026년 4월        "이 5개 상품을    "통과 상품을      "이 데이터를
미국에서 떠오르는    7대 Kill         30점 만점         바차트로
DTC 카테고리       Criteria로        스코어카드로       시각화"
5개 + 출처"        평가"             환산"
```

**실행:**
1. Perplexity: "2026년 4월 미국 향 DTC 시장에서 검색량 30%+ 상승 카테고리 5개 + 각 출처 URL"
2. Perplexity 결과 → Claude에 붙여넣기 + Master Prompts §1 (Kill Criteria) 실행
3. PASS 상품 → Claude로 30점 스코어카드 환산
4. Claude의 점수 데이터 → Gemini로 차트 생성 ("Make a bar chart of the following data: ...")

**결과**: 90분 노가다 → 25분 완성

## 🎯 레시피 13: A/B 광고 닫힌 학습 루프 (W9~W12)

```
[Spark Ads CSV: 5종 카피 4시간 데이터]
              │
              ▼
[Claude: 가장 잘된 카피의 "왜?" 분석 + 약점 식별]
              │
              ├───→ "v2 카피 3종 자동 생성 (강점 강화 + 약점 보정)"
              │                        │
              │                        ▼
              │              [4시간 더 테스트]
              │                        │
              ├───────── 결과 누적 (각 라운드별 학습 노트)
              │
              ▼
[3 라운드 후] [전체 데이터 → 페르소나·훅·CTA 베스트 패턴 추출]
              │
              ▼
[다음 분기 전체 광고 = 학습 결과 기반 새 5종]
```

**실행 (라운드 1회 = 1시간 + 4시간 광고비 + 분석)**:

**라운드 1** ([§20 5종](./02_Master_Prompts.md) 카피 → 4시간 운영):
| 카피 | CTR | CPC | CVR | 메모 |
|---|:--:|:--:|:--:|---|
| A (Pain) | 1.2% | $0.85 | 0.8% | |
| B (Benefit) | 0.8% | $1.20 | 1.1% | |
| C (Curiosity) | 2.1% | $0.45 | 0.6% | 훅 강함 |
| D (Social Proof) | 1.6% | $0.65 | 1.4% | CVR 1위 |
| E (Authority) | 0.5% | $1.80 | 0.3% | KILL |

**라운드 2 자동 생성**:
```text
"위 5종 데이터를 분석해 다음 v2 카피 3종 생성:
- D의 'Social Proof' CVR 강점 + C의 'Curiosity' 훅 결합
- E의 'Authority'는 폐기
- 각 v2에 'D의 어떤 요소를 가져왔는지' + 'C의 어떤 요소를 가져왔는지' 메모
- Self-Critique 1회 적용 — 약점 재작성"
```

**라운드 3** = D+C 결합형 가장 잘된 1개를 강화한 v3 3종.

**누적 학습 노트** (분기말):
```yaml
best_hook: "POV: 매일 X 겪는 너" (Curiosity-led, 페르소나 1 적합)
best_cta: "Grab yours before this 24h sale ends" (Social Proof-led, 페르소나 3 적합)
```
