## 📖 이론 파트 (0:10 - 0:40)

### 1. 오가닉 트래픽이 유료 광고를 이기는 이유

초보자가 50달러로 페이스북 광고를 돌리면 100% 돈을 허공에 날립니다. 왜일까요? 스토어에 방문해도 **리뷰나 사람들의 반응(Social Proof)이 없기 때문**입니다. 그래서 우리는 **커미션을 20~30%까지 파격적으로 주고**, 대신 물건을 팔아줄 '용병'들을 먼저 모읍니다. 내 돈은 한 푼도 들지 않습니다 — 그들이 팔아올 때만 돈을 주면 되니까요.

### 2. 마이크로 인플루언서 (10K ~ 50K) 공략법

팔로워 100만 명인 메가 인플루언서에게 DM을 보내면 읽지도 않습니다. 우리의 타겟은 **팔로워 1만~5만 명 사이**, 이제 막 채널이 크고 있어서 "협찬"에 목말라 있는 마이크로 인플루언서입니다. 이들에게 무료 샘플을 던져주고 영상을 뽑아내는 것이 핵심입니다.

#### 📌 크리에이터 점수표 (4축 평가 — 30점 만점)

| 평가 축 | 1점 | 3점 | 5점 |
|---|---|---|---|
| **팔로워 적합도** | <5K 또는 >100K | 5K~10K 또는 50K~100K | **10K~50K (스위트 스팟)** |
| **평균 조회수** | <1K | 1K~10K | >10K |
| **참여율** (좋아요 ÷ 조회수) | <3% | 3~7% | >7% |
| **니치 톤 매칭도** | 우리 제품과 무관 | 일부 연관 | 완벽 일치 |

> **합산 17점 미만 → 드롭, 17~22점 → DM 발송, 23점 이상 → 우선 컨택**

### Step 2. AI 초개인화 DM 대량 생성 (30분)

이번 주 핵심은 **2단계 프롬프트 파이프라인**입니다. 채널 분석 → DM 생성을 분리해 품질을 극대화합니다.

#### Step 2-A. 채널 분석 ([Master Prompts §6](./02_Master_Prompts.md))
1. [02_Master_Prompts.md §6 크리에이터 채널 분석 프롬프트](./02_Master_Prompts.md)를 엽니다.
2. 크리에이터의 최근 영상 3개(제목 + 캡션 + 톱2 댓글)를 입력합니다.
3. JSON 결과를 받습니다 (`tone`, `interests`, `specific_detail`, `fit_score`).

#### Step 2-B. 채널-개인화 DM 생성 ([Master Prompts §7](./02_Master_Prompts.md))
1. [02_Master_Prompts.md §7 채널-개인화 AI DM 풀 프롬프트](./02_Master_Prompts.md)를 엽니다.
2. §6의 JSON 결과를 그대로 입력 변수에 붙여넣습니다.
3. 결과물 DM은 **`specific_detail`을 첫 줄에 자동 인용**해 템플릿 냄새가 사라집니다.
4. 50명분을 순식간에 생성한 뒤 §3 CRM 시트의 `DM_Draft` 컬럼에 저장합니다.

#### 📌 DM 안티-차단 규칙 (위반 시 계정 정지 위험)

| 규칙 | 사유 |
|---|---|
| 동일 IP에서 1시간당 **15건 이하** 발송 | 틱톡/IG의 봇 감지 임계치 |
| 이모지 **2개 이하** | 3개 이상 = 스팸 신호 |
| **첫 DM에 링크 금지** | 즉시 차단 트리거 |
| `@멘션` 금지 | 푸시 알림 스팸으로 분류 |
| "collaboration / partnership / promo" **첫 문장 금지** | 키워드 필터에 즉시 걸림 |
| 발송 간 **랜덤 30~120초 간격** | 인간형 패턴 시뮬레이션 |
| 본인 프로필에 사이트 URL·연락처 미리 등록 | 상대가 검증할 수 있어야 답장률 ↑ |

구글 스프레드시트를 열고 아래 11컬럼 스펙을 그대로 만듭니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q: 50명한테 보냈는데 한 명도 답장이 안 오면 어쩌죠?**
> 보통 100명에게 보내면 5명 정도 답장이 옵니다 (5% 응답률). 응답이 없다면 둘 중 하나입니다 — 본인의 인스타/틱톡 계정 프로필이 사기꾼처럼 비어있거나, DM 첫 줄이 너무 스팸 같기 때문입니다. 프로필을 프로페셔널하게 꾸미고, AI가 써준 "상대방 칭찬" 문구를 첫 줄에 꼭 넣으세요.

**Q: 샘플을 보내줬는데 먹튀(영상 안 올림)하면 어떡하나요?**
> 먹튀는 제휴 마케팅의 피할 수 없는 **'사업 비용(Cost of doing business)'**입니다. 5명에게 샘플(원가 15달러)을 보내서 총 75달러를 썼는데, 그중 1명이 영상을 올려서 조회수 100만이 터지고 매출 3천 달러가 발생하면? 압도적인 이득입니다. 먹튀에 감정 상하지 말고 모수를 늘리세요.

## 🩺 응답률 5% 미달 시 진단 트리

50명 발송 후 회신이 2건 미만이면 아래 4단계 의사결정 트리를 돌립니다.

```
응답률 < 5%
  │
  ├─ Q1. 내 프로필이 비어 있는가? (사진/링크/소개 없음)
  │     YES → 🔧 프로필부터 정비 (다른 모든 액션 무의미)
  │
  ├─ Q2. DM 첫 줄에 §6 specific_detail이 인용되었는가?
  │     NO  → 🔧 §6→§7 파이프라인 누락. DM 재생성
  │
  ├─ Q3. 발송 시간대가 미국 오후 6~10시(태평양)인가?
  │     NO  → 🔧 발송 시간 시프트
  │
  └─ Q4. 점수표 합산 23점 이상의 크리에이터에 발송했는가?
        NO  → 🔧 니치 미스매치. Step 1 검색 쿼리 재선정
```
